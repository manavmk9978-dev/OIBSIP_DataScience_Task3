# Car Price Prediction using Machine Learning

## Objective

The objective of this project is to predict car selling prices using Machine Learning techniques.

## Tools and Technologies Used

- Python

- Google Colab

- Pandas

- NumPy

- Matplotlib

- Scikit-learn

## Algorithm Used

Linear Regression

## Dataset

The dataset contains:

- Car Name

- Year

- Present Price

- Kms Driven

- Fuel Type

- Selling Type

- Transmission

- Owner

- Selling Price

## Steps Performed

1. Imported required libraries

2. Loaded the dataset

3. Converted categorical data into numerical values

4. Split the dataset into training and testing data

5. Trained the Linear Regression model

6. Predicted car prices

7. Evaluated model performance

## Outcome

The model successfully predicted car prices based on car features.

## Conclusion

Machine learning can help estimate car prices accurately using historical car data.
